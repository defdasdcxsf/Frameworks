# Frameworks
Open Source mainly AI casting mod for Gorilla Tag.


Frameworks is a clean, professional casting and rig management tool for Gorilla Tag. It provides advanced camera controls, rig smoothing, and overhead data visualization.

### features
- basically the same as every other casting mod but not paid so thats a +
- self rig smoothing (aka you can self cast)
- customizable nametag shiz
- more stuff idk
- sexy ass gui
